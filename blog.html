What Is The Document Object Model (DOM)?

The Document Object Model (DOM) is a programming API for HTML and XML documents. It defines the logical structure of documents and the way a document is accessed and manipulated.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the context of the DOM (Document Object Model) where a single event listener is attached to a parent element, rather than to multiple child elements. This parent event listener can listen for events that occur on its child elements, even if the child elements are dynamically added later. The technique takes advantage of event bubbling, where events triggered on child elements propagate (or "bubble up") to their parent elements.

It is useful for:

  • Efficiency
  • Dynamic Elements
  • Simpler Code
How do you manipulate an element's attributes and styles using the DOM?

To manipulate attributes and styles in the DOM:

Attributes:

  • Get: element.getAttribute('attribute')
  • Set: element.setAttribute('attribute', 'value')
  • Remove: element.removeAttribute('attribute')

Styles:

  • Change style: element.style.propertyName = 'value'
  • Example: element.style.backgroundColor = 'blue'
